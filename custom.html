BigBlueButton

BigBlueButton is an alternative for teachers, tutors and profs around the world to Zoom.

If jitsi, doens't suit your needs you can use BigBlueButton

Features

Why should I use BigBlueButton?!

  • Virtual Whiteboard with collab feature
  • Sharing Webcam
  • Chat
  • Record your lessons
  • Easy to use

Ressources

For BigBlueButton (best practice) you'll need the following ressources:

  • CPU with at least 4 Cores a 2.5GHz
  • Around 16 to 32GB RAM
  • Storage (if you want to record your sessions)
    • 15min recording = approx 100MB Storage
  • Public IP
    • prefered IPv4 & IPv6 dedicated to that host
  • Internet Connection
    • prefered 1Gbit/s symetric on the server
  • Ubuntu 16.04 (yes. really.)

Jitsi

Jitsi is an open source video/audio chat solution.

Features

Why should I use Jitsi?!

  • Sharing Webcam
  • Sharing Screen
  • Chat
  • Easy to use

Ressources

For Jitsi you'll need the following ressources:

  • Small up to 5 persons:
    • 1 Core (2.5GHz min)
    • 2 GB RAM
    • approx 20GB HDD/SSD
    • Public IP
  • Medium up to 10 persons:
    • 2 Core (2.5GHz min)
    • 4 GB RAM
    • approx 20GB HDD/SSD
    • Public IP
  • Large up to 25 persons:
    • 4 Core (2.5GHz min)
    • 8GB RAM
    • approx 20GB HDD/SSD
    • Public IP
  • For all instances we're using latest Ubuntu LTS OS & Docker Container
